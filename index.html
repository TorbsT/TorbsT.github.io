How does searching work?

You can search for project titles, programming languages/tools, years, descriptions, as well as availability on other sites.
Example: "abductees, unity, 2021, game jam, github, itch, released, -team"

Criteria are separated using comma.
Inclusive search will show projects that satify ANY of the criteria.
Non-inclusive search will show projects that satisfy ALL of the criteria.
Putting a dash "-" in the start of a criterium specifies it should NOT be satisfied.
