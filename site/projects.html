How does searching work?

You can search for project titles, programming languages/tools, years, descriptions, as well as availability on other sites.
Example: "Abductees, Unity, 2021, game jam, github, itch.io"

Criteria are separated using comma.
Inclusive search displays projects that include ANY of the criteria.
Non-inclusive search displays projects that include ALL of the criteria.
